# Get-Smtp4DevContainerIp

## SYNOPSIS
Gets the IP of the smtp4dev container

## SYNTAX

```
Get-Smtp4DevContainerIp [<CommonParameters>]
```

## DESCRIPTION
Inspect the smtp4dev Container and return the IP Address of the first network.

## EXAMPLES

### EXAMPLE 1
```
Get-Smtp4DevContainerIp
```
